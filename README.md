# Student Task Manager

## Project Overview
This project is a web-based task manager designed to help students organise coursework, deadlines, and daily tasks more effectively.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Due dates
- Task persistence using localStorage
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript

## Project Aim
To develop a simple and effective productivity tool for students while demonstrating software design, development, and testing skills.
